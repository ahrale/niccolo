# 포토 필터 — 설계 (2026-07-04, 오너 승인)

아이폰용 사진 필터·보정 웹앱. 로컬/개인용 저사양 스펙, 지인 아이폰에서 링크로 사용 가능해야 함.

## 결정 사항
- **공유 방식**: GitHub Pages 호스팅 (오너 선택)
- **저장 방식**: Web Share API로 사진 앱 저장, 미지원 시 개별 다운로드 폴백 (오너 선택)

## 구조
- 의존성 없는 단일 `index.html` (vanilla JS + canvas). 프레임워크·빌드 없음.
- 사진 입력: `<input type="file" accept="image/*" multiple>` — iOS Safari가 HEIC를 JPEG로 자동 변환하므로 HEIC 라이브러리 불필요. 디코드 실패 파일은 건너뛰고 안내.
- 메모리: 미리보기는 최대 1400px 다운스케일 canvas로 보관, 원본 해상도 디코드는 내보내기 시 한 장씩.

## 필터·보정
- 필터 5종: Clarendon / Gingham / Juno / Lark / Valencia — 각각 밝기·대비·채도·색온도·페이드 수치 조합으로 근사.
- 미리보기와 내보내기가 동일한 픽셀 파이프라인(`applyPipeline`) 공유 — Safari `ctx.filter` 호환 이슈 회피, 미리보기=결과물 보장. 채널별 점 연산은 256-LUT.
- 보정 슬라이더: 밝기 / 대비 / 채도 / 선명하게(상하좌우 언샤프 마스크).
- Crop: 정규화 좌표(0..1) 저장, 프리셋 자유·1:1·4:5·16:9, 포인터 드래그 이동·코너 리사이즈(비율 고정 지원).

## 일괄 적용
- "전체 적용" = 현재 사진의 필터+슬라이더를 모든 사진에 복사. crop은 사진별 구도가 달라 제외.

## 내보내기
- 사진별: 원본 재디코드 → crop 영역 canvas → 파이프라인 → `toBlob('image/jpeg', 0.9)` → `{원본명}_edited.jpg`.
- `navigator.share({files})`로 여러 장 한 번에 공유시트 → 사진 앱 저장. `canShare` 실패 시 `<a download>` 개별 다운로드.

## 범위에서 뺀 것 (YAGNI)
회전·원근 보정, 필터 강도 슬라이더, 되돌리기 히스토리, ZIP 다운로드. 필요해지면 추가.
